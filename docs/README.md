# Corpus de Credibilidad, Sesgo y Riesgo en Información Financiera (México)

Proyecto de Servicio Social orientado a la construcción de un corpus etiquetado de textos financieros en español (México), clasificados según:

- Credibilidad
- Sesgo informativo
- Nivel de riesgo

## Objetivo

Construir un dataset estructurado que pueda ser utilizado para análisis descriptivo y futuras aplicaciones en Procesamiento de Lenguaje Natural (NLP).

## Estructura del Proyecto

- docs/ → Documentación metodológica
- data/ → Datos crudos y procesados
- scripts/ → Scripts de procesamiento
- notebooks/ → Análisis exploratorio
- reports/ → Entregables formales
