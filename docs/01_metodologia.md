# Metodología

## Fases del Proyecto

1. Definición conceptual y diseño del esquema de etiquetado.
2. Diseño de la estructura del dataset.
3. Recolección piloto de textos.
4. Etiquetado manual inicial.
5. Ajuste de criterios tras análisis de consistencia.
6. Escalamiento del corpus.
7. Análisis descriptivo utilizando Python (pandas).

## Herramientas

- Git y GitHub para control de versiones.
- Python (pandas) para análisis.
- Archivos CSV para almacenamiento estructurado.
- Markdown para documentación metodológica.
